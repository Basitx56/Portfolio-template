# 📸 Photography Portfolio Website

A responsive **photographer portfolio website** built with pure HTML, CSS, and JavaScript — developed as part of the **IICT (Introduction to Information & Communication Technology)** course at FAST NUCES Islamabad.

---

## 🌐 Live Preview

> Open `index.html` in any browser to view the site locally.

---

## ✨ Features

- Smooth preloader animation on page load
- Responsive navigation with mobile hamburger menu
- Hero section with photographer introduction
- Photo **Gallery** section
- **About** section with signature and quote
- **Services** section
- **Portfolio** section showcasing work
- Login page with form validation
- Scroll-to-top button
- Custom fonts (Gordita & Recoleta)
- Fully responsive design for all screen sizes

---

## 🛠️ Tech Stack

| Technology | Usage |
|------------|-------|
| HTML5 | Page structure |
| CSS3 | Styling & animations |
| JavaScript | Interactivity & login validation |

No frameworks. No libraries. Pure vanilla web development.

## 🚀 How to Run

1. Clone or download the repository
2. Open `index.html` in your browser

```bash
git clone https://github.com/Basitx56/<portfolio-template>.git
cd <portfolio-template>
# Open index.html in your browser
```

No installation or build step required!
